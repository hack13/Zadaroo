XML Importable Content

XML Importable content that can be imported with a V1 Style viewer, the only still actively developed V1 Viewer that has this feature still is Singularity Viewer. You will need to go into the file menu and select "Import XML" and then select the XML document from the extracted zip contents and import, however you will need to make sure you are in an area where you have Rez permissions.

Clothing & Body

Clothing and Body Content are textures that you can upload in almost any viewer, and apply to clothing layers in your avatar customization panel. Some clothing does have XML importable parts for prim parts of the clothing, you will need to use a viewer like Singularity Viewer to import that content.

Animations & Poses

Animations are importable via any viewer, including modern viewers. This content type is animations that allow you to animate your avatar in ways outside of the stock movement animations.
