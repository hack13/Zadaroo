OARs

OAR Files are region archives. Please pay close attention as not all of these files are for OpenSimulator, some of these files were created for Aurora-Sim(Current fork named WhiteCore-Sim). If you try to load a WhiteCore-Sim file into OpenSimulator it can cause a bunch of serialization issues! You have been warned.

Load From Site:

  1. Open your region's console
  2. Copy the hyperlink to the OAR file
  3. Run the following command: load oar https://files.zadaroo.com/oars/name_of_oar_file.tgz

Download Option:

  1. Download the OAR file you want
  2. Record where you placed the file
  3. Run the following command: load oar /path/to/file/name_of_oar_file.tgz

Disclaimer: I am not resposible for any lost inventory or inventory issues you may have. I am simply providing these files and I do not claim to have much technical knowlege of this proceedure. Although I did test it and it worked fine for me with no problems at all.
